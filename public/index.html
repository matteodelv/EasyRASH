    EasyRASH Project Info

    EasyRASH is a simple web application that let you easily manage conferences and academic events by focusing on the Peer Review process for articles written in RASH. Click on the Help button on top to discover all the features of EasyRASH, its usage and icon meanings.

    EasyRASH Help

    Here you can find some information about how to use EasyRASH and how to review an article or manage an event, icon meanings and a small description of the interface.

    Interface

    Navigation Bar

    Located on top just below the title, contains key elements to interact with events and papers. On the left side, there is the "Mode Toggle", which let every reviewer to enter in "Annotator Mode" to add annotations to a paper as a review. Furthermore, there are two hidden buttons to let every chair manage an event: in fact, they will only appear when the logged in user is one of the chairs of the currently selected conference. The first button show the Management Panel for a conference, where every chair can edit its info, and add co-chairs or reviewers; the second button, instead, is used by every chair to assign reviewers to the currently open paper.

    On the right side, there are an Help button and an Info button.

    Main Area

    It is the biggest part of the interface and it is used to show papers, for both reading and review. In some specific conditions, based on paper status and user's role, annotations will also be displayed. In addition, papers can be shown in RASH or Springer LNCS layout. A sidebar affixed on the right will help you scroll through the different sections and keep track of where you're at.

    Sidebar

    Located on the left, the sidebar is the main way to navigate between papers and conferences. On top, it shows info about the currently logged in user. Below there is the Conferences Selector, where users can select specific events and get papers from that event, or they can create new conferences. The last section, divided in "Authored By Me" and "Pending Reviews", shows a list of all the articles where the logged in user is author and all the papers that haven't been reviewed yet, since the user is reviewer for them.

    Note: This last section can be considered as a summary of the work related to the user, no matter which conference is or is not selected.
